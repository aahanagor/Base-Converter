# Base Converter
## Overview
Interactive C++ console application that converts numbers between Decimal, Binary, and Hexadecimal formats. 
## Purpose
Allows users to input a number in either Base 10 (Decimal) or Base 16 (Hexadecimal) and instantly converts it to the other number systems. The program runs in a loop until the user chooses to quit.
## Features
- Convert Decimal → Binary
- Convert Decimal → Hexadecimal
- Convert Hexadecimal → Binary
- Convert Hexadecimal → Decimal
- Automatic leading zero removal for clean output
- Menu-driven console interface
- Continuous execution until user selects QUIT
## Made Using
- C++
